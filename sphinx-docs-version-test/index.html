cloudify-gcp-plugin documentation¶

The Cloudify GCP plugin allows users to Cloudify to manage resources on Google Cloud Platform. See Node Types for the supported resource types.

Plugin Requirements¶

  • Python versions: * 2.7.x
  • GCP account

Compatibility¶

The GCP plugin uses the official Google API Python Client.

Account Information¶

The plugin needs access to your GCP auth credentials (via the gcp_config parameter) in order to operate (but see Manager GCP Config if using a manager).

Terminology¶

  • region refers to a general geographical area, such as “Central Europe” or “East US”.
  • zone refers to a distinct area within a region. Zones are usually referred to as ‘{region}-{zone}, i.e. ‘us-east1-b’ is a zone within the reigon ‘us-east1’.
